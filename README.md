## Cross-SDK Evidence Command

Run the end-to-end Go vs NodeJS sqlite evidence workflow with one command:

```bash
./scripts/run_cross_go_nodejs_sqlite_evidence.sh
```

Optional environment variables:

- LCP_CROSS_DATASET_COUNT (default: 120)
- LCP_CROSS_ARTIFACT_DIR (default: autogenerated under .artifacts/)

## Cross One-by-One Command

Run Go (memory+sqlite) vs JavaScript (IndexedDB) one-by-one parity checks with one command:

```bash
./scripts/run_cross_go_javascript_onebyone.sh
```

Optional environment variables:

- LCP_CROSS_GO_JS_ARTIFACT_DIR (default: autogenerated under .artifacts/)
- LCP_CROSS_GO_JS_SKIP_BUILD (set to 1 to skip JavaScript build)

## Cross One-by-One Command (Cloud)

Run Go (memory+cloud-storage) vs JavaScript (IndexedDB) one-by-one parity checks:

```bash
./scripts/run_cross_go_javascript_cloud_onebyone.sh
```

Optional environment variables:

- LCP_CROSS_GO_JS_CLOUD_ARTIFACT_DIR (default: autogenerated under .artifacts/)
- LCP_CROSS_GO_JS_SKIP_BUILD (set to 1 to skip JavaScript build)
- LCP_CROSS_GO_CLOUD_STORAGE_URI (default: gs://aiptesting.firebasestorage.app/lcp)
